Смотреть результаты через инспектор HTML

Все дополнительные атрибуты, классы и тд ставить между img и src, иначе может что-то не сработать

  1. Стандартная картинка без ничего

  2. чтобы не изменять картинку нужно добавить между <img и src атрибут noSource - принудительно не изменять, у картинки noSorce автоматически будет удалён

  3. svg картинка (она никак не изменяется)

  4. если путь к картинке начинается не с "img/", то ничего не заменять, Если начинается с img, но по такому пути картинки нет, то замена на конструкцию произойдёт всё равно

  5. если у картинки стоит атрибут sourceClass="some-class", то значение арибута станет классом для source, а из картинки атриубт будет удалён

  6. data-src - добавляет data- к srcset

  7. non-resize картинка (к назвванию картинки добавлена приставка "nr_", вместо трёх сорсов подставится 1, только для вебп, без медиазапросов

  8. у картинки есть класс и id (проверка работы регулярки)

  9. в редакторе тег разделён переносом строк для каждого атрибута (проверка работы регулярки)
    выглядит вот так: 

    					<img
						class="asdas"
						src="img/about_us/menu.jpg"
						width="200"
						height="100"
						alt="asdas"
					/>
